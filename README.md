# Overview
   This is miniature chat tools, Only to learn server development 

### Feature
    
   - Login 
   - Logout 
   - Chat 

    
# Ready

### Install environment

   1. Install Python3.8
    
    sudo apt-get update
    sudo apt-get install python3.8
    
   2. Install python packages 
   
    pip install -r requirements.txt

# Run MeChat server
    python start_server.py
